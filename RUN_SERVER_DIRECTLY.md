# Run Server Directly (Bypass npm)

Since you have disk space issues but `node_modules` exists, you can run the server directly:

## Quick Start

```bash
cd ecommerce-backend
node server.js
```

This bypasses npm and runs the server directly using the existing dependencies.

## To Stop
Press `Ctrl+C` in the terminal

## Note
The server will work exactly the same as `npm run dev`, just without nodemon auto-restart.
